Course Module

Month 1: Foundations of Data Science

Week 1-2: Introduction to Data Science & Python

Day 1 (3 hours):
  • Introduction to Data Science: Concepts, Applications in Fintech
  • Data Science Life Cycle (Data Collection, Cleaning, Analysis, Interpretation)
  • Overview of Python for Data Science
Day 2 (3 hours):
  • Python Setup & Basics (Data types, Variables, Loops, Functions)
  • Hands-on: Basic Python Exercises

Week 3-4: Data Handling with Python

Day 1 (3 hours):
  • Introduction to Python Libraries: NumPy and Pandas
  • Data Manipulation with Pandas: Series, DataFrames, Indexing, and Slicing
Day 2 (3 hours):
  • Importing/Exporting Data (CSV, JSON, Excel)
  • Hands-on: Manipulating DataFrames, Handling Missing Data

Month 2: Data Preprocessing and Exploration

Week 5-6: Data Wrangling & Preprocessing

Day 1 (3 hours):
  • Data Cleaning: Handling Missing Values, Duplicates, Outliers
  • Data Transformation: Rescaling, Encoding Categorical Variables
Day 2 (3 hours):
  • Hands-on: Cleaning and Preprocessing a Real Dataset
  • Exploratory Data Analysis (EDA): Techniques & Visualization

Week 7-8: Visualization and Feature Engineering

Day 1 (3 hours):
  • Data Visualization with Matplotlib & Seaborn
  • Hands-on: Plotting Charts (Bar, Line, Scatter)
Day 2 (3 hours):
  • Feature Engineering: Feature Selection & Creation
  • Hands-on: Feature Engineering Techniques

Month 3: Machine Learning Basics & Final Project

Week 9-10: Introduction to Machine Learning

Day 1 (3 hours):
  • Introduction to Machine Learning: Supervised vs. Unsupervised Learning
  • Overview of Algorithms: Linear Regression, Classification
Day 2 (3 hours):
  • Hands-on: Implementing Linear Regression using Scikit-learn
  • Evaluating Models: Accuracy, Precision, Recall

Week 11-12: Unsupervised Learning & Final Project

Day 1 (3 hours):
  • Introduction to Unsupervised Learning: Clustering (K-Means, Hierarchical)
  • Hands-on: Clustering Algorithms using Scikit-learn
Day 2 (3 hours):
  • Project Work: Applying Concepts to a Real-world Dataset
  • Course Wrap-up & Career Advice in Data Science
